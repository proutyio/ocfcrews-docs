# Categories

## Overview

The **Categories** collection stores content categories used for organizing items in the shop or other content areas. Each category has a human-readable title and an auto-generated URL slug. Categories are publicly readable but only manageable by admin and editor users.

## Configuration

| Property | Value |
|---|---|
| **Slug** | `categories` |
| **Admin Group** | Shop |
| **Use As Title** | `title` |

## Fields

| Name | Type | Required | Description |
|---|---|---|---|
| `title` | `text` | Yes | The display name of the category (max 100 characters). |
| `slug` | `text` | No | URL-friendly slug, auto-generated by Payload's built-in `slugField()`. Useful for constructing category-based URLs and filtering. |

## Access Control

| Operation | admin | editor | All Other Roles | Unauthenticated |
|---|---|---|---|---|
| **Create** | Yes | Yes | No | No |
| **Read** | Yes | Yes | Yes | Yes |
| **Update** | Yes | Yes | No | No |
| **Delete** | Yes | Yes | No | No |

- **Read**: Fully public -- returns `true` for all users (authenticated or not).
- **Create / Update / Delete**: Restricted to users with `admin` or `editor` roles.

## Hooks

This collection does not define any hooks.

## Relationships

| Related Collection | Relationship Type | Description |
|---|---|---|
| (Referenced by other collections) | -- | Categories can be referenced by any collection that needs content categorization (e.g., shop items, posts). |
