# OCFCrews Documentation

Technical documentation for the [OCFCrews](https://github.com/proutyio/ocfcrews) crew scheduling platform, built with [Docusaurus](https://docusaurus.io/).

## Setup

```bash
npm install
```

## Local Development

```bash
npm start
```

Starts a local dev server at `http://localhost:3000/docs/`. Changes are reflected live.

## Build

```bash
npm run build
```

Generates static content into the `build` directory.

## Deployment

The docs site is served from the main OCFCrews application at `/docs/`. Build output should be deployed to the app's `public/docs/` directory or hosted separately.
